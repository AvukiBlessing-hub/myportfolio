# Avuki Blessing Isaac — Portfolio

## Folder Structure
```
portfolio/
├── index.html
├── style.css
├── script.js
├── assets/
│   └── profile.jpg       ← ADD YOUR PHOTO HERE
├── cv/
│   └── Avuki_Blessing_Isaac_CV.pdf  ← ADD YOUR CV HERE
└── README.md
```

## Setup
1. Drop your photo into `assets/` — name it `profile.jpg` (square or portrait crop works best)
2. Drop your CV into `cv/` — name it `Avuki_Blessing_Isaac_CV.pdf`
3. Open `index.html` in your browser to preview

## Deploy (free)
- **Netlify**: drag the folder to netlify.com/drop
- **Vercel**: push to GitHub, import on vercel.com
- **GitHub Pages**: enable in repo Settings → Pages
